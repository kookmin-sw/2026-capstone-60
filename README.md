# Intertalk Landing Page

국민대학교 2026 캡스톤디자인 60팀 Intertalk 소개 페이지입니다.

## Tech

- Vite
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
